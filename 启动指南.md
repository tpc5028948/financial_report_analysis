## 如何访问前端界面

### 方法一：直接打开HTML文件

1. **找到前端文件**：
   - 文件位置：`E:\traeuse\A\financial_report_analysis\frontend\index.html`

2. **打开方式**：
   - 方法1：双击该文件直接在浏览器中打开
   - 方法2：右键点击文件，选择"打开方式" → 选择浏览器

### 方法二：使用本地服务器（推荐）

如果直接打开HTML文件遇到跨域问题，可以使用Python启动本地服务器：

```bash
# 在前端目录下启动服务器
cd E:\traeuse\A\financial_report_analysis\frontend
python -m http.server 8000
```

然后在浏览器中访问：http://localhost:8000

## 常见问题及解决方案

### 1. 前端无法连接到后端
- **问题**：前端显示"无法连接到服务器"
- **解决方案**：检查后端服务是否正在运行，确保地址为 http://localhost:5000

### 2. 文件上传失败
- **问题**：上传文件后显示错误
- **解决方案**：确保文件格式为 PDF、Excel 或 HTML，文件大小不要过大

### 3. 分析结果不显示
- **问题**：点击分析后没有结果
- **解决方案**：检查浏览器控制台是否有错误信息，确保网络连接正常
